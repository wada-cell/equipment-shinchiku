1. ログイン方法

  1. スマートフォンのブラウザ（Chrome, Safari等）でアプリのURLを開く
  2. 管理者から発行されたメールアドレスとパスワードを入力
  3. 「ログイン」ボタンをタップ
NOTE パスワードを忘れた場合は管理者に連絡してリセットしてもらってください。 ブックマーク（お気に入り）やホーム画面に追加しておくと、次回から簡単にアクセスできます。

2. パスワードを変更する

  1. 画面下の「マイページ」タブをタップ
  2. 「パスワードを変更する」をタップ
  3. 現在のパスワードと新しいパスワード（4文字以上）を入力
  4. 「変更する」をタップして完了
NOTE セキュリティのため、初回ログイン後にパスワードを変更することをおすすめします。

3. ホーム画面の見方

カテゴリ選択「着工備品」と「現調セット・その他備品」の2つから選びます
期限超過アラート返却期限を過ぎた備品がある場合、画面上部に表示されます
貸出状況現在貸出中の備品が一覧で表示されます

4. 備品を借りる

  1. ホーム画面でカテゴリを選択
  2. 借りたい備品をタップ（キーボックス等は中に入って番号を選ぶ）
  3. 備品詳細画面で「貸出申請する」をタップ
  4. 以下の項目を入力する：
    • 使用現場・お客様名
    • 持ち出し日
    • 持ち出し時間（1時間刻みのボタン or 直接入力）
    • 返却予定日
    • 返却予定時間
    • 使用目的
  5. 「貸出申請する」で完了
NOTE 「使用中」と表示されている備品は、他の人が借りています。リクエスト機能（5章）で交渉できます。

5. 備品を返却する

  1. ホーム画面の貸出状況、またはマイページから借りている備品をタップ
  2. 備品詳細画面で「返却する」をタップ
  3. 確認画面で「OK」を押して完了
注意事項のある備品
  • キーボックス → ナンバーを「0000」に戻してから返却
  • 一眼レフカメラ → SDカードの中身を空にしてから返却
※ 備品詳細画面に返却注意事項が赤字で表示されます。必ず確認してください。

6. 使用リクエストを送る

  1. 使用中の備品の詳細画面を開く
  2. 「使用リクエストを送る」をタップ
  3. 使いたい日時とメッセージを入力して送信

現在借りている人に通知が届きます。直接連絡を取り合って、返却タイミングを調整してください。

6.5 備品を予約する

  1. 備品の詳細画面を開く
  2. 「予約」セクションの「＋ 予約する」をタップ
  3. 開始日時・終了日時・現場名・使用目的を入力
  4. 「予約する」をタップして完了
NOTE 同じ期間に別の予約や貸出がある場合はエラーになります。予約のキャンセルは予約一覧の「取消」から行えます。

6.6 貸出カレンダーを見る

  1. ホーム画面の「貸出カレンダー」をタップ
  2. 「＜」「＞」で月を切り替え
  3. 日付をタップすると、その日の貸出一覧が表示されます
NOTE 黒い点がある日は貸出中の備品があります。スプレッドシートの月表示の代わりに活用してください。

7. 通知を確認する

  • 画面下の「通知」タブをタップ
  • ヘッダー右上のベルアイコンからも確認できます
  • 未読の通知がある場合、バッジ（数字）が表示されます
返却リマインド返却予定時刻が近づくと通知されます
使用リクエスト他のスタッフからリクエストが届いた場合に通知されます

8. 【管理者】備品を追加・削除する

  1. マイページ →「管理者メニュー」→「備品の追加・編集」
  2. 「＋ 備品を追加」をタップ
  3. 以下を入力する：
    • カテゴリ（着工備品 / 現調セット等）
    • 備品名（例：キーボックス）
    • 番号（例：No.10）※ 同じ名前+番号違いは自動でグループ化
    • 保管場所（例：イナバ倉庫）
    • 返却時の注意（例：ナンバーを0000に戻す）
    • メモ（自由入力）
  4. 「追加する」で完了

削除する場合は、備品一覧の各備品の横にある「削除」をタップしてください。

9. 【管理者】スタッフを登録する

  1. マイページ →「管理者メニュー」→「ユーザー管理」
  2. 「＋ スタッフを追加」をタップ
  3. 名前・メールアドレス・初期パスワード・権限（スタッフ/管理者）・部署を入力
  4. 「追加する」で完了

登録後、メールアドレスとパスワードをスタッフに伝えてください。

10. よくある質問

Q. パスワードを忘れた管理者に連絡してリセットしてもらってください
Q. パスワードを変更したいマイページ →「パスワードを変更する」から変更できます
Q. 借りたい備品が使用中「使用リクエスト」を送って、借りている人と交渉してください
Q. 新しい備品を追加したい管理者メニューの「備品の追加・編集」から追加できます
Q. カテゴリを増やしたい管理者メニューの「カテゴリ管理」から追加できます
Q. ログアウトしたいマイページの一番下に「ログアウト」ボタンがあります
